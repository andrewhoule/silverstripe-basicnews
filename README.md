Simple News Module
==================

## Author
* Andrew Houle
* http://andyhoule.com

## Requirements
* SilverStripe minimum version 3.0+.

## Installation
* Add the news folder to your site's root.
* Run http://yoursite.com/dev/build/?flush=all
* Create a new news holder page and sub news pages.
